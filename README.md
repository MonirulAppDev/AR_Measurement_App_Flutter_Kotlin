# Area Measurement AR

A premium Flutter application for measuring areas and distances in the real world using Augmented Reality (AR).

## Features

- **AR Measurement**: Tap to place points on detected surfaces to measure distances.
- **Real-time Feedback**: Visual indicators and distance readouts as you measure.
- **Premium UI**: Sleek, modern design with glassmorphism effects and smooth interactions.
- **Cross-Platform**: Built with Flutter for a consistent experience on supported AR devices.

## Getting Started

### Prerequisites

- Flutter SDK
- A device with AR support (ARCore for Android, ARKit for iOS)

### Installation

1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run` on your AR-enabled device

## Technologies Used

- **Flutter**: UI Framework
- **ar_flutter_plugin_2**: AR capabilities
- **vector_math**: For geometric calculations
